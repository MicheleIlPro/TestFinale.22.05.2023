Cos'è Js quali sono le sue caratteristiche

Js (JavaScript) è un linguaggio di programmazione molto utilizzato, È caratterizzato da una tipizzazione debole, praticamente possiamo assegnare ad una variabile qualsiasi tipo, poichè possiamo creare una variabile senza specificarne il tipo. let i = 1;
È interpretato ed orientato agli ogetti e viene eseguito direttamente dal browser, praticamente viene eseguito così come viene visualizzato il codice.
Oltre a tutto questo, javascript ha un sacco di librerie e framework utilizzabili (tipo React e Angular) che già esistono o che probabilmente verranno inventati nel tempo.

Cos'è CSS quali sono le sue caratteristiche

CSS, sta per Cascade Style Sheet ed è un linguaggio di stile, NON di programmazione.
Ci permette di mettere uno stile al nostro file HTML con forme, bordi, colori, grandezze e di fare animazioni semplici.
Lavora utilizzando classi, id, e tag generici che utilizzeremo nel file html.
Il css inoltre ha un sistema di priorità, dove siamo più specifici avremo più priorità detta basicamente.

Cos'è HTML quali sono le sue caratteristiche

HTML, sta per Hyper Text Markup Language, NON di programmazione
Serve a definire la base di una pagina web, che integrata con css potrà essere modificata esteticamente.
Si basa su degli elementi chiamati Tag, Tag semantici che hanno ognuno una funziona propria. oltre a tutto ciò che è contenuto nel HEAD il più utilizzato è il DIV.
Possiamo inserire link nella pagina con il tag A, oppure immagini, audio o video con IMG, AUDIO e VIDEO.
Con h1-6 possiamo creare dei titoli e con P dei paragrafi.

Cos'è Angular e quali sono le su principali caratteristiche?

Angular è un framework che si basa sul linguaggio di programmazione TypeScript (un Javascript fortemente tipizzato)
È basato su un architettatura a componenti che ci permette di creare un componente con ng g c nome_componente e renderlo riutilizzabile in caso di necessità ed è OTTIMO per Header e Footer ripetuti. È anche possibile utilizzare il Routing che ci mette a disposizione, facendoci caricare anticipatamente le pagine tra cui switcheremo.

In cosa si differenziano i linguaggi tipizzati e non , quindi TS e JS

Un linguaggio tipizzato dà errore quando non assegni un tipo ad una variabile creata/inizializzata.
Un linguaggio non tipizzato ti permette di assegnare ad una variabile un valore senza specificare direttamente cosa gli stiamo dando.
